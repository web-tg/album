##移动web相册——大图预览
#####实现功能：相册大图预览
#####运用知识：
    1.zepto.js中swipeLeft，swipeRight，tap等事件;
    2.html5中canvas代替img绘制图片(开启GPU渲染)和图片预加载 ;
    3.css3动画bounceIn,bounceInRight,bounceInLeft;
    4.css3动画事件webkitAnimationEnd;
